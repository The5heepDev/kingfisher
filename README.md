#Solus Wallpaper Generator

#How to install:
	1. Extract the Wallpaper folder into your home directory.
	2. Run install.sh
	3. Run 'wallpaper' for a new wallpaper. If you don't like the wallpaper, just rerun the file!
	4. Enjoy your wallpapers!
